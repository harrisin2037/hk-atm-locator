# Scraping ATM data from all banks in Hong Kong

## Installing
````
python3 -mvenv ./env
source ./env/bin/activate
pip3 install -r requirements.txt
````

## Formatting code
Example:
````
yapf -i hkbea.py
````

## Running the scraper
Example:
````
python3 hkbea.py
````
